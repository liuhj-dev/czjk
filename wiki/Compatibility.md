# 设备兼容性

本文档列出无驱动录音录像系统**支持的设备**、**操作系统**和**硬件要求**。

---

## 📷 支持的摄像头设备

### UVC 兼容摄像头

**标准**: USB Video Class (UVC) 1.0, 1.1, 1.5

#### ✅ 完全支持的设备类型

- USB 网络摄像头（Logitech, Microsoft, HP 等）
- 高拍仪（大部分现代型号）
- USB 文档相机
- 部分 USB 采集卡
- 部分 USB 显微镜摄像头

#### ❌ 不支持的设备

- 仅提供厂商私有驱动的设备
- 需要特殊 SDK 的设备（如部分工业相机）

## 🎤 支持的音频设备

### UAC 兼容麦克风

**标准**: USB Audio Class (UAC) 1.0, 2.0

#### ✅ 完全支持的设备类型

- USB 麦克风
- USB 耳机（带麦克风）
- 部分 USB 音频接口
- 部分 USB 话放

#### ❌ 不支持的设备

- 仅提供厂商私有驱动的设备
- 专业音频接口（如部分 Focusrite, PreSonus 型号可能需要专有驱动）


```bash
sudo apt-get update
sudo apt-get install -y \
    libopencv-dev \
    ffmpeg \
    v4l-utils \
    pulseaudio \
    libpulse-dev
```

**用户权限配置**:
```bash
sudo usermod -a -G video $USER
sudo usermod -a -G audio $USER
# 注销并重新登录使权限生效
```

**设备检查**:
```bash
# 列出所有视频设备
v4l2-ctl --list-devices

# 列出所有音频设备
arecord -l
```

## 🖥️ 硬件要求

### 最低配置

| 组件 | 要求 | 说明 |
|------|------|------|
| CPU | Intel Core i3 或 AMD 等效（2 核 2.0 GHz+） | 能运行，但性能有限 |
| 内存 | 2 GB RAM | 仅适合标清录制 |
| 硬盘 | 100 MB（不含录制文件） | SSD 推荐 |
| USB | USB 2.0+ | USB 3.0 更推荐 |
| 显卡 | 集成显卡（Intel HD 4000+） | - |

**适用场景**: 测试、学习、低分辨率录制（VGA/720p）

---

### 推荐配置

| 组件 | 要求 | 说明 |
|------|------|------|
| CPU | Intel Core i5/i7 或 AMD Ryzen 5/7（4 核 3.0 GHz+） | 性能良好 |
| 内存 | 4-8 GB RAM | 适合大部分场景 |
| 硬盘 | SSD 1 GB+（不含录制文件） | 提高读写速度 |
| USB | USB 3.0+ | 减少带宽瓶颈 |
| 显卡 | 独立显卡（NVIDIA GTX 1050+ 或 AMD RX 560+） | 支持硬件加速 |

**适用场景**: 生产环境、1080p 录制、多任务

---

### 4K 录制配置

| 组件 | 要求 | 说明 |
|------|------|------|
| CPU | Intel Core i7/i9 或 AMD Ryzen 7/9（8 核 3.5 GHz+） | 必须高性能 CPU |
| 内存 | 16 GB RAM | 4K 视频处理需要大内存 |
| 硬盘 | SSD（读写速度 500 MB/s+） | NVMe SSD 推荐 |
| USB | USB 3.0+ | 4K 视频需要高带宽 |
| 显卡 | NVIDIA RTX 3060+ 或 AMD RX 6700XT+ | 必须支持硬件编码 |

**适用场景**: 专业视频制作、监控系统、高质量录像

## 🔧 已知问题和限制

### 摄像头问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 设备无法打开 | 设备被占用 | 关闭其他使用摄像头的程序（如 Skype、Zoom） |
| 画面卡顿 | USB 带宽不足 | 降低分辨率或帧率，使用 USB 3.0 |
| 颜色异常 | 白平衡错误 | 在摄像头属性中调整，或使用 `setVideoParameters()` |
| 无法识别 | 驱动不兼容 | 确保设备支持 UVC 标准，尝试其他 USB 接口 |

### 音频问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 无音频输入 | 麦克风未选中 | 在系统设置中选择正确的输入设备 |
| 音频延迟 | 缓冲区过大 | 减小 `TargetDataLine` 缓冲区大小 |
| 杂音/回声 | 降噪未启用 | 启用软件降噪或使用带降噪的麦克风 |
| 音量过低 | 增益不足 | 调整麦克风增益（系统设置或代码） |

### 录制问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 音视频不同步 | 编码延迟 | 使用相同的 timebase，降低分辨率 |
| 文件过大 | 比特率过高 | 降低视频或音频比特率 |
| 无法播放 | 编码参数错误 | 使用标准编码参数（H.264/AAC） |
| 录制中断 | 磁盘空间不足 | 确保足够的磁盘空间，设置 `maxFileSize` |
